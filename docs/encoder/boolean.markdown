Boolean Encodings
-----------------

### `BOOLEAN_8BITS_ENUM_FIXED`

The input values are encoded as the 8-bit constants `0x00` (false) and `0x01`
(true).

#### Options

None

#### Conditions

None

#### Examples

The input value `true` is encoded as:

```
+------+
| 0x01 |
+------+
```

The input value `false` is encoded as:

```
+------+
| 0x00 |
+------+
```
